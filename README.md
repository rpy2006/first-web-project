Features

· Modern & Responsive Design: Fully responsive layout that works on all devices
· Loading Screen: Animated loading screen with progress bar

· Services Showcase: Display of professional services with embedded YouTube videos

Getting Started

Prerequisites

No special requirements needed. The website runs directly in any modern web browser.

Installation

1. Clone the repository:

```
git clone https://github.com/rpy2006/first-web-project.git
```

1. Navigate to the project directory:

```
cd first-web-project
```

1. Open index.html in your web browser.
